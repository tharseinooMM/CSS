Display CSS

display မှာ ၄မျိုးရှိတယ်။ block, inline, inline-block, none
blockက အပြည့်ယူတယ်။ ဒါကြောင့် ထိန်းချုပ်လို့ရတယ်။
inlineက တစ်နေရာစာဘဲယူတယ်။ ဒါကြောင့် ထိန်းချုပ်လို့မရ။
တစ်ခါတလေကျရင် inlineကိုလည်း ထိန်းချင်တယ်။ ဒါပမေယ့် block ကြီးဘဲလည်းမဖြစ်စေချင်ဘူးဆိုရင် inline-block ကိုသုံးချင်တယ်
noneကတော့ ဘာမှမလုပ်တာကိုပြောတယ်။
